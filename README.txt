
# Сайт коллекции бутылочек

## Как обновлять коллекцию
1. Открывайте Google Таблицу
2. Добавляйте новые строки
3. Сайт обновится автоматически

## Важное
Google Таблица должна быть:
- опубликована в интернете
- доступна для просмотра всем

## Загрузка на Vercel
1. Зайдите на https://vercel.com
2. Нажмите Add New → Project
3. Перетащите ZIP-архив
4. Нажмите Deploy
